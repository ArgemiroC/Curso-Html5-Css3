Glossário Para Meus Estudos

Termos Gerais

    👉 Back-end

    É o termo usado para descrever a parte da programação de um website ou aplicação que é responsável por lidar com os processos internos e as funcionalidades que não são diretamente visíveis para o usuário. Isso inclui coisas como banco de dados, lógica de negócios e integração com outros sistemas. O back-end geralmente é escrito em linguagens de programação como Python, Ruby, Java, entre outras. Ele cuida da lógica da aplicação e gerencia os dados que são mostrados no front-end para o usuário final.

    👉 Copyright

    Copyright é um conjunto de leis que garantem aos autores e editores o direito exclusivo de reproduzir, distribuir e exibir suas obras por um período de tempo determinado. Ele também garante que os direitos autorais só podem ser usados com permissão do proprietário da obra. O símbolo de copyright é " © ", geralmente acompanhado pelo ano de publicação e o nome do proprietário do copyright. Por exemplo: © 2022, João Silva.

    👉 Front-end

    É o termo usado para descrever a parte da programação de um website ou aplicação que é responsável por lidar com a interface e a interação com o usuário. Isso inclui coisas como HTML, CSS e JavaScript, que são usadas para construir e formatar a página, bem como gerenciar eventos e interações do usuário. O front-end se preocupa com como a aplicação se parece e se comporta para o usuário final.

    👉 Full Stack

    É o termo usado para descrever um desenvolvedor ou programador que tem habilidades e conhecimentos para trabalhar tanto no front-end quanto no back-end de um website ou aplicação. Isso significa que eles são capazes de trabalhar com tecnologias de interface do usuário, como HTML, CSS e JavaScript, bem como com tecnologias de back-end, como banco de dados, lógica de negócios e integração com outros sistemas. Um desenvolvedor full stack pode criar uma aplicação completa, desde o design e layout até a funcionalidade interna e armazenamento de dados.

    👉 Index

    É geralmente o arquivo principal de uma pasta ou diretório. O arquivo "index.html" é o arquivo padrão que é utilizado como página inicial, mas pode ser utilizado outros tipos de arquivos como "index.php" ou "index.asp" dependendo da configuração e do objetivo da aplicação.

    👉 Tag

    É usada para indicar o início e o fim de uma seção de conteúdo, como uma parágrafo, cabeçalho, lista, etc. Por exemplo, a tag <p> indica o início de um parágrafo e a tag </p> indica o fim do parágrafo

Termos HTML5

    👉 <!DOCTYPE html>

    É uma declaração obrigatória no início de um documento HTML que especifica qual versão do HTML está sendo usada. Isso permite que os navegadores e outros processadores de HTML saibam como interpretar e exibir o documento corretamente.

    👉 <html lang="pt-br">

    Define a linguagem do página.

    👉 <head> </head>

    A tag é usada em HTML para delimitar o início da seção de cabeçalho de um documento. A seção de cabeçalho geralmente contém informações sobre a página, como o título da página (definido com a tag <title>) e metadados (definidos com a tag <meta>), além de links para folhas de estilo (CSS) e scripts (JavaScript). Essas informações são usadas pelos navegadores, mecanismos de busca e outros sistemas para entender e exibir o conteúdo da página de maneira apropriada.

    👉 UTF-8

    É um formato de codificação de caracteres que significa Formato de Transformação Unicode 8-bit. permite que a página web exiba caracteres de diferentes idiomas e scripts, incluindo caracteres acentuados e outros símbolos especiais. É importante incluir a declaração <meta charset="UTF-8"> na seção <head> do HTML para garantir que o navegador entenda que o documento está sendo codificado em UTF-8, caso contrário, pode haver problemas de exibição de caracteres específicos.

    👉 "viewport"

    Informa que o site vai ocupar toda tela do dispositivo

    👉 link: favicon

    Indica a imagem que vai aparecer no navegador

    👉 link: css

    Indica o caminho para o arquivo CSS (Estilos)

    👉 <meta> </meta>

    A tag é usada em HTML para fornecer informações adicionais sobre a página que não são exibidas diretamente para os usuários. Ela é geralmente incluída dentro da tag <head> do documento HTML e pode conter informações como descrição da página, palavras-chave, autor, tipo de conteúdo, caracteres de codificação, entre outras. Essas informações são usadas por mecanismos de busca, navegadores e outros sistemas para entender e exibir o conteúdo da página de maneira apropriada.

    👉 <title> </title>

    A tag é usada em HTML para definir o título de uma página. Ela é geralmente encontrada dentro da tag <head> do documento HTML e é usada para fornecer uma descrição sucinta do conteúdo da página. O título é exibido na aba do navegador ou no topo da janela do navegador. Ele também é usado pelos mecanismos de busca para exibir o título da página nos resultados de pesquisa.

    👉 <style> <style>

    É um termo geral que se refere ao conjunto de regras e propriedades que definem a aparência de um elemento HTML.

    👉 <header> </header>

    A tag é usada para definir uma seção que contém informações de cabeçalho para um documento ou seção

    👉 <nav> </nav>

    A tag é usada para identificar uma seção que contém elementos de navegação. Ela é usada para indicar que a seção contém links para outras páginas ou áreas do site. É geralmente usada no cabeçalho ou rodapé de uma página web, mas pode ser usada em qualquer lugar do conteúdo da página.

    👉 <body> </body>

    É usada para definir o conteúdo principal de uma página web. Contém todo o conteúdo de um documento HTML, como texto, imagens, links, tabelas, listas, etc. Ele é o principal container de todos os elementos visuais do site, e geralmente vem depois da seção <head>, que contém metadados e outras informações sobre o documento. Também pode ter atributos como bgcolor, text, background, link, alink, vlink e onload, etc. Esses atributos são usados para personalizar a aparência e comportamento do conteúdo do corpo da página.

    👉 <main> </main>

    A tag é usada para definir uma seção no final de um documento ou seção. É comumente usado para conteúdo de rodapé, como informações de contato, créditos de autoria ou links de navegação. Ele é geralmente usado em conjunto com outras tags de estrutura, como <header> e *ltsection>, para criar uma estrutura de documento completa e organizada.

    👉 <footer> </footer>

    A é usada para definir uma seção no final de um documento ou seção. É comumente usado para conteúdo de rodapé, como informações de contato, créditos de autoria ou links de navegação. Ele é geralmente usado em conjunto com outras tags de estrutura, como <header> e <section>, para criar uma estrutura de documento completa e organizada.

    👉 <h1> até <h6>

    As tags são usadas em HTML para definir os títulos e subtítulos de uma página. Elas são usadas para estruturar o conteúdo da página e ajudar os usuários a entender a hierarquia do conteúdo.

    👉 <p> </p>

    Abre e fecha um parágrafo

    👉 <br>

    Quebra a linha e inicia abaixo

    👉 <hr>

    A tag é utilizada para criar uma linha horizontal, ou "ruler" em inglês, no documento. Ela é usada para separar conteúdo em diferentes seções ou para criar uma estrutura visual no documento. A tag <hr> é uma tag vazia, ou seja, não possui conteúdo ou fechamento, e é geralmente usada como uma tag de formatação.

    👉 <strong> </strong>

    É uma tag que é usada para dar ênfase ao texto, geralmente tornando-o em negrito. Ele indica que o texto dentro da tag deve ser exibido com uma maior importância ou relevância em relação ao texto ao seu redor.

    👉 <abbr> </abbr>

    A tag é usada para marcar uma abreviação ou sigla. Ela pode ser usada para incluir o significado da abreviação ou sigla como atributo "title", permitindo que os usuários entendam o conteúdo mesmo que não conheçam a abreviação. Além disso, essa tag pode ser estilizada com CSS para destacar visualmente as abreviações no texto.

    👉 <table> </table>

    A tag é usada para criar tabelas de dados. Ele é composto por várias outras tags, como <tr> (table row), <th> (table header) e <td> (table data), que são usadas para criar as linhas, cabeçalhos e celulas respectivamente. As tabelas são muito úteis para exibir informações estruturadas, como dados financeiros, estatísticas, etc. Além disso, as tabelas também podem ser estilizadas com CSS para torná-las mais atraentes e fáceis de ler.

    👉 <tr> </tr>

    A tag é usada para criar uma linha em uma tabela. Ele é geralmente utilizado dentro de uma tag <table> e contém várias células de dados (tags <td>) que formam uma linha na tabela. Essa tag é usada para definir uma linha de dados na tabela, que pode conter uma ou mais células de dados.

    👉 <th> </th>

    A tag é usada para criar um cabeçalho de tabela. Ela é geralmente usada dentro de uma tag para definir uma célula de cabeçalho em uma linha de tabela. O conteúdo dentro da tag <th> é geralmente exibido como negrito e centralizado, e é usado para fornecer informações sobre os dados contidos nas células de dados (tags <td>) relacionadas. Além disso, a tag <th> tem atributos adicionais, como "scope" que pode ser usado para indicar se o cabeçalho é um cabeçalho de coluna, linha ou grupo de dados.

    👉 <td> </td>

    A tag é usada para criar uma célula de dados em uma tabela. Ela é geralmente utilizada dentro de uma tag <tr> para definir uma célula de dados em uma linha de tabela. O conteúdo dentro da tag <td> é geralmente exibido como texto normal e alinhado à esquerda, e é usado para fornecer informações sobre os dados contidos na tabela. Além disso, a tag <td> pode ser estilizada com CSS para melhorar a aparência da tabela e torná-la mais fácil de ler.

    👉 "scope"

    O atributo "scope" na tag <th> em HTML é usado para indicar a área de uma tabela para qual o cabeçalho específico se aplica. Ele pode ter três valores diferentes: 👉"row" indica que o cabeçalho se aplica às células na linha atual 👉"col" indica que o cabeçalho se aplica às células na coluna atual 👉"rowgroup" ou "colgroup" indica que o cabeçalho se aplica a um grupo de linhas ou colunas respectivamente. Esse atributo é útil para acessibilidade, pois ajuda os leitores de tela a compreender a estrutura da tabela e a relação entre os cabeçalhos e os dados.

    👉 <img>

    A tag é usada no HTML para inserir imagens em uma página web. Ela é composta por uma tag de abertura <img≶ e, geralmente, inclui atributos como "src" (fonte), que especifica o caminho para a imagem, e "alt" (texto alternativo), que fornece uma descrição da imagem para usuários com dificuldades visuais. Exemplo: <img src="image.jpg" alt="Descrição da imagem">

    👉 <address> </address>

    A tag é usada no HTML para indicar informações de contato ou endereço. Geralmente, é usado para exibir informações de contato de um site, como endereço físico, número de telefone ou endereço de e-mail. O conteúdo dentro da tag <address> é geralmente exibido em um estilo de fonte diferente, como itálico, para diferenciá-lo do conteúdo principal da página.

Termos CSS3

    👉 'background-color'

    É uma propriedade do CSS (Cascading Style Sheets) que é usada para definir a cor de fundo de um elemento HTML. Ele pode ser definido como uma cor sólida, um gradiente ou uma imagem de fundo.
